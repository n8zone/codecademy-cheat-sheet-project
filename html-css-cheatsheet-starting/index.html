Typography

Property Description Values
font-size Controls the size of the font. Can use keywords, em, rem, percentages, or pixel values.
font-style Defines how much the text is slanted. normal, italic, oblique
font-variant Defines which glyph to use for each letter. normal, small-caps
font-weight Defines the weight of the text. keywords, integers(100-900)
font Shorthand property for font-style font-variant font-weight font-size line-height font-family N/A
letter-spacing Defines the spacing between characters of a block of text. normal, pixels, em
line-height Defines the height of a single line of text. normal, pixels, em, unitless (relative to font-size)
text-align Defines the horizontal alignment of text. left, right, center, justify
text-decoration Defines whether or not the text is underlined. none, underline
text-indent Defines indentation of the first line of text. Numeric unit values, 0 for no indent.
text-overflow Defines how text that overflows its container behaves clip, ellipsis
